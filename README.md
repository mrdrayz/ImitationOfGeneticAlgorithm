# Команда установки зависимостей

npm install

# Команда для запсука

npm start

# Цель работы

Написанная программа, реализует генетический алгоритм для решения задачи оптимизации с изменяющимися параметрами.
Найти минимум функции:
f(x) = (x-1.5)^3 + 3
Вариант A) Использовать целочисленное кодирование.
Вариант Б) Использовать вещественное кодирование.
